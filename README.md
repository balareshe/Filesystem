# 用户档案系统

AstrBot 插件 — 在 QQ 内管理用户的档案信息，支持 **全局统一存储、跨群互通、域名信息查询**。

📦 GitHub：https://github.com/balareshe/Filesystem

## 功能

| 命令 | 说明 |
|------|------|
| `/绑定账号` | 绑定 QQ 号创建档案（**仅限群聊**） |
| `/查询档案` | 查询档案（管理员可 @ 查他人） |
| `/绑定网站 域名` | 智能模式：自动识别网站标题，类型默认为「博客」 |
| `/绑定网站 名称 类型 域名` | 手动指定网站信息 |
| `/更新网站 名称 类型 域名` | 更新已绑定的网站信息 |
| `/解绑网站` | 解绑网站，基础档案保留 |
| `/查询域名` | 查询绑定域名的实时信息（WHOIS、延迟、地区等） |
| `/删除档案 @用户` | 删除用户档案（仅白名单管理员） |
| `/评价` | 基于多维度数据对网站进行综合评分（10项/5分制） |
| `/hda` | 显示所有命令帮助 |

## 特点

### 全局档案
档案跟随 QQ 号，**不区分群**。在任意群绑定后，所有群都能查到同一份档案。

### 跨群互通
- 在 A 群 `/绑定账号` → B 群也能 `/查询档案`
- 在 B 群 `/绑定网站` → A 群也能看到网站信息
- 修改、解绑等操作全局生效

### 私聊支持
除 `/绑定账号` 外，其他所有命令均支持私聊使用：
- `/查询档案` — 私聊查自己的档案
- `/查询域名` / `/评价` — 私聊查域名信息与评分
- `/绑定网站` / `/更新网站` / `/解绑网站` — 私聊管理网站

### 域名查询
`/查询域名` 可获取域名实时信息：

```
网站标题 '摆烂的小站'
响应状态 '200'
响应延迟 '328ms'
服务器 'nginx'
服务器地区 '中国 广东 深圳'
注册商 'Alibaba Cloud'
注册时间 '2023-01-15'
过期时间 '2027-01-15'
最近更新 '2026-01-16'
域名有效期 '333天'
```

- 网站标题、服务器、响应状态 → 实时抓取
- 服务器地区 → IP 归属地查询（ip-api.com）
- 注册商、注册/过期/更新时间 → WHOIS 实时查询

### 智能模式
`/绑定网站 域名` 支持智能识别：
- 自动抓取网站标题作为网站名称
- 类型默认为「博客」
- 如自动识别失败，可使用手动模式

### 固定编号
每个用户绑定账号时自动分配全局唯一的编号，后续不会改变。

### 网站评价
`/评价` 基于 10 个维度对绑定的网站进行综合评分：

```
📋 网站评价 - 摆烂的小站
⭐⭐⭐⭐✨ 4.8/5.0  (优势9项 基于10/10项数据)

非常优秀的网站！各方面表现出色，维护状态极佳。

详细评分：
✅ 域名有效期充裕(>1年)
✅ 证书有效期充足(183天)
✅ 响应速度快(475ms)
✅ 网站标题完整有意义
✅ 服务器信息可见(nginx)
✅ 服务器地区可查(日本 东京都)
✅ 注册商信息可见(成都垦派科技有限公司)
⚪ 域名注册超过6个月
✅ SSL证书有效(183天)
✅ 支持HTTPS且正常响应
```

**评分维度（每项 0.5 分，共 5 分，拿不到数据的项不影响评分）：**

| 维度 | 数据来源 |
|------|---------|
| 域名有效期 | WHOIS / 免费 API |
| 域名过期时间 | WHOIS / 免费 API |
| 响应速度 | HTTP 请求 |
| 标题质量 | 网页抓取 |
| 服务器信息 | 响应头 |
| 服务器地区 | IP 归属地 |
| 注册商信息 | WHOIS / 免费 API |
| 域名年龄 | WHOIS / 免费 API |
| SSL证书 | SSL 直查 / 免费 API |
| HTTPS 状态 | HTTP 请求 |

## 网站类型

`博客` `企业` `个人` `商城` `论坛` `其他`

## 权限

- **普通成员**：绑定账号、查询档案、绑定/更新/解绑网站（仅本人）
- **白名单管理员**：删除档案、@ 查他人档案（在插件配置 `admin_ids` 中填写 QQ 号）
- 管理员权限**只认白名单**，群主/群管理员不算

## 数据存储

数据位于 AstrBot 内置 KV 存储中，按 QQ 号全局存储（`profile_{QQ号}`），每个用户仅一份档案。

## 依赖

- `aiohttp` — 网页抓取与 WHOIS / IP 归属地查询
- `python-whois` — WHOIS 域名信息查询

## 支持平台

- QQ（aiocqhttp）
